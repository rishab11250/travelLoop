# TravelLoop - Ultimate Travel Planning Application

TravelLoop is a comprehensive travel planning platform that helps users organize their trips, build itineraries, and share experiences with a community.

## 📂 Repository Structure

- **`frontend/`**: React + Vite application for the user interface.
- **`server/`**: Node.js + Express backend (APIs and database management).

## 🚀 Getting Started

### 1. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### 2. Backend Setup
```bash
cd server
npm install
npm start
```

## 🛠️ Tech Stack
- **Frontend**: React, React Router, Zustand, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express
- **APIs**: Axios for frontend-backend communication

## 📁 Key Features (Frontend)
- **Trip Management**: Create, edit, and view personal trips.
- **Itinerary Builder**: Section-based daily itinerary planning.
- **Checklists & Notes**: Stay organized with trip-specific tasks and reminders.
- **Community**: Share and explore travel plans from other users.
- **Export to PDF**: Save your itineraries as PDF for offline access.
